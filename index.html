Cara Mendapatkan Google Apps Script URL

          
Fokus ke URL yang berakhiran /exec.

        

        
      

      

        

          

            
1Buat Spreadsheet Baru

            
https://docs.google.com/spreadsheets/u/0/

          

          

            
2Pilih Blank Spreadsheet

            
Klik: Blank (Spreadsheet kosong). Boleh ganti nama sesuai bisnis Anda.

          

          

            
3Buka Apps Script dari Spreadsheet

            
Di Spreadsheet: Extensions → Apps Script. Ini memastikan script terhubung langsung ke Spreadsheet Anda.

          

          

            
4Copy Script (Otomatis)

            
Klik tombol di bawah untuk menyalin script yang sudah disesuaikan dengan field form Anda.

            
          

          

            
5Tempel Script

            
Paste ke Code.gs (hapus isi lama jika ada), lalu klik Save.

          

          

            
6Deploy Web App

            
Deploy → New deployment → Web app
Execute as: Me, Access: Anyone

          

          

            
7Salin URL /exec

            
Salin URL yang berakhiran /exec dari deployment Web App.

          

          

            
8Tempel ke Form Builder + Tes Koneksi

            
Masukkan ke kolom Google Apps Script URL, lalu klik Tes Koneksi sampai status Berhasil Terhubung.
